# java-pass-manager

Hello, this is my java password manager written in java of course. 
It features multiple things like the ability to assign password information to websites like a normal password manager but even more than that, 
it has the ability to assign and record security questions and answers that you might need for certain websites. 

It runs locally on all windows computers and does not require or use an internet connection so it is very safe and all local and open source. 

Furthermore, the password manager also allows you to save your username and search by website name instead of username which makes it much easier to pull out info. 

There is a .exe release of the program in releases to try out and demo.
